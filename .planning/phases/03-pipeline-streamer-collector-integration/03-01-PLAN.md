## Phase Goal

**As a** platform operator, **I want to** stream live DCGM GPU telemetry from the CSV through the message queue into PostgreSQL, **so that** every reading is durably stored exactly once even when producers and collectors run concurrently.

This plan delivers the shared mapping contract that the Collector slice depends on.

## Artifacts this phase produces (this plan)

- Package `github.com/ajitg/vantage/pkg/models`
- `type GpuMetric struct` — fields: `GpuID string`, `Timestamp time.Time`, `MetricName string`, `Value float64`, `Device string`, `ModelName string`, `Hostname string`, `Container string`, `Pod string`, `Namespace string`, `LabelsRaw string`
- `func FromProto(msg *pb.TelemetryMessage) (GpuMetric, error)`
- `const InsertSQL string` — 11-column `INSERT ... ON CONFLICT (gpu_id, metric_name, timestamp) DO NOTHING`
