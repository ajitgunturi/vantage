## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator env → Config.FromEnv | `MQ_CONSUME_CREDIT` is operator-supplied; a non-numeric or non-positive value must not corrupt the credit default. |
| disconnect path → Store.Requeue | Re-enqueued lease values cross back into shared store state under concurrency. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01.1-03 | Denial of Service | `MQ_CONSUME_CREDIT` parse | medium | mitigate | `strconv.Atoi` + `err == nil && n > 0` guard rejects negative/zero/garbage; default 20 stands. (Per-consumer credit clamping of the on-wire value is enforced by the engine, Plan 03.) |
| T-01.1-04 | Denial of Service | `RingStore.Requeue` under buffer-full | medium | mitigate | Requeue is bounded by ring capacity; full-ring eviction drops the newest entry (counted in `dropped`) — no unbounded growth, no amplification on redelivery storms. RESEARCH Pitfall 7: size MQ_BUFFER_SIZE >> credit x consumers. |
| T-01.1-SC | Tampering | dependencies | low | accept | No new packages; stdlib `sync`/`strconv`/`os` only (RESEARCH Package Legitimacy Audit: none). |
